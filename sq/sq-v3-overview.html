Architect
계획 수립 & 에이전트 선택
수정 목적을 분석해 plan.md와 agents.txt 자동 생성. runner / scorer / keyword_fixer / reporter 중 필요한 에이전트를 선택.
항상: runner · scorer · keyword_fixer · reporter
🏃
Wave A
runner — 키워드 수집 & 검색 API 배치 호출
PHASE 1 DB 쿼리로 4가지 유형 키워드 수집 (ZERO_RESULT 150 + ETC_TYPE 100 + NO_SYNONYM 100 + POSSIBLE_FALLBACK 100) + 웹서칭 50개 → 검색 API 배치 호출 → results/*.json 저장.
ZERO_RESULT 150개 ETC_TYPE 100개 NO_SYNONYM 100개 POSSIBLE_FALLBACK 100개 웹서칭 50개
📊
Wave B
scorer — v2 채점 + fixable 필드 추가
results/*.json을 v2 채점 체계(100점/6항목)로 채점하고, v3 신규 필드 fixable / fix_type / fix_priority를 포함한 scores.json 생성.
🔧
Wave C — 신규
keyword_fixer — 컨텍스트 조회 & SQL 생성
scores.json의 fixable 키워드마다 PHASE 2 컨텍스트 쿼리(F-1~F-6) 실행 → fix_search_keywords.sql (BEGIN/COMMIT 트랜잭션 포함) + fix_preview.md 생성. 현재 read-only이므로 SQL 파일만 생성, 실행하지 않음.
⚠️ SQL 파일만 생성 — read-only 계정
📄
Wave D
reporter — reporter-v3로 리포트 생성
reporter-v3 사용. §7 수정 계획 요약(fix_preview.md 기반) + §8 SQL 미리보기 30줄 섹션이 추가된 리포트 3종 생성.

키워드 수집 전략 — v2 → v3 변화
수집 유형sq / sq-v2sq-v3변경 이유
웹서칭 (트렌드) 250개 50개 수정 효과보다 현황 파악 목적 → 비중 축소
DB — ZERO_RESULT rank 상위 통합 150개 (result_product_count=0) 수정 즉시 검색 결과 생김 → 최우선
DB — ETC_TYPE — 100개 (keyword_type=ETC) must 필터 없음 → 타입 변경으로 개선 가능
DB — NO_SYNONYM — 100개 (SAME 관계 없음) 동의어 추가로 Tier 1 결과 다양화
DB — POSSIBLE_FALLBACK — 100개 (발골사전 미등록) 카테고리·브랜드 매핑 추가로 FALLBACK 해소
합계 500개 500개 (중복 제거 후)
diagnosis_codes → fix_type 매핑
diagnosis_code파이프라인 실패 지점수정 테이블fix_typepriority
ZERO_UNRECOGNIZED _classify_tokens 매칭 실패 → 발골 전부 없음 search_keywords INSERT INSERT_KEYWORD HIGH
ZERO_NO_MATCH Tier 1 & 2 모두 0건 search_keyword_relations SIMILAR INSERT ADD_SIMILAR LOW
ETC_TYPE keyword_type=ETC → must 필터 없이 should만 search_keywords type UPDATE + 매핑 INSERT FIX_KEYWORD_TYPE MEDIUM
NO_SYNONYM _expand_keywords SAME 0건 → Tier 1 원본만 search_keyword_relations SAME INSERT ADD_SYNONYM MEDIUM
POSSIBLE_FALLBACK extracted 없음 → should 전체 검색 search_category/brand_keywords INSERT FIX_CATEGORY_MAPPING / FIX_BRAND_MAPPING MEDIUM
BRAND_MAPPING_WRONG 잘못된 brand.id 필터 search_brand_keywords brand_id UPDATE FIX_BRAND_MAPPING MEDIUM
CATEGORY_MAPPING_WRONG product_type/midtype 불일치 search_category_keywords UPDATE FIX_CATEGORY_MAPPING MEDIUM
LOW_RELEVANCE OpenSearch 분석기 이슈 코드 수정 필요 NOT_FIXABLE NONE
SOLDOUT_EXPOSED is_sold_out 필터 없음 코드 수정 필요 NOT_FIXABLE NONE
